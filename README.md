# Telegram File-Link Converter Bot

This is a Telegram bot built with Python that converts files into links and converts links back into downloadable files.

## Features

- Convert uploaded files into shareable links
- Convert links into downloadable files
- Simple and easy to use

## How It Works

1. Send a file to the bot to receive a link
2. Send a supported link to the bot to receive the file

## Requirements

- Python 3.x
- Dependencies listed in requirements.txt
